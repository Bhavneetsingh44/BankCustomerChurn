# 🏦 Bank Customer Churn Prediction

Predicting which bank customers are likely 
to leave using Machine Learning.

## Progress
✅ Day 1 - Data Cleaning (9,988 customers)
✅ Day 2 - Exploratory Data Analysis


## Key Findings So Far
- 🇩🇪 Germany churns 2x more than France & Spain
- 📦 3-4 products = 83-100% churn rate
- 👴 Age 50-60 = highest risk group (56% churn)
- 💰 High balance customers churning more!

## Dataset
10,000 European bank customers
Target: Will customer leave? (Yes/No)

## Tech Stack
Python | Pandas | Scikit-learn | Streamlit
